#########################################
Vera C. Rubin Observatory Prompt Products
#########################################

**Anticipated start of Alert Production**: February 2026

Prompt Products include the alerts, catalogs, and images generated by real-time Rubin Science Pipelines processing of observations obtained with the LSST Camera.

.. important::

   This website is currently under development.

**Data Policy:** Some Prompt Products are public and some proprietary: see :doc:`/access/index` for details.
